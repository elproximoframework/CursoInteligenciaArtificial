Manejo de Errores: El Porqué

El Plan de Contingencia: `try / except / finally`

try:

Poner aquí el código que podría fallar.

Ej: `abrir_archivo('datos.csv')`

except:

Se ejecuta solo si hay un error en el `try`.

Ej: `print("El archivo no existe")`

finally:

Se ejecuta siempre, con o sin error.

Ej: `cerrar_recursos()`

La diferencia entre un prototipo frágil y un sistema de producción robusto.

Manejo de Excepciones

Elegancia y Eficiencia en Python

Comprensión de Listas

Forma Tradicional (Bucle `for`)


cuadrados = []
for n in lista:
    cuadrados.append(n * n)

3-4 líneas, más verboso.

Forma "Pythónica"


cuadrados = [n * n for n in lista]

1 línea, conciso, más rápido.

9 / 12
Comprensión de Listas

El Poder en una Sola Línea

Lista Original

[1, 2, 3, 4]

Comprensión de Lista

[n * n for n in lista]

Nueva Lista

[1, 4, 9, 16]

Se lee casi como en inglés: "Dame el cuadrado de n para cada n en la lista".

Ejemplo de Comprensión

Comunicación con el Mundo Exterior

Manejo de Archivos

La Forma Segura: `with open(...)`


with open('mi_archivo.txt', 'r') as f:
    for linea in f:
        print(linea)

La estructura `with` garantiza que el archivo se cierra automáticamente, incluso si ocurren errores.
