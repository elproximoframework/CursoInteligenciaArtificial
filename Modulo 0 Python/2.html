Operadores: Las Herramientas

Cómo Operamos con Nuestros Datos

Aritméticos

Suma (+), Resta (-), Multiplicación (*), División (/). Los cálculos de siempre.

Comparación

Mayor que (>), Menor que (<), Igual a (==). Hacen preguntas y devuelven True/False.

Lógicos

Y (`and`), O (`or`), No (`not`). Combinan las preguntas para crear lógica compleja.

Control de Flujo: La Encrucijada

Tomando Decisiones con `if`, `elif`, `else`

if temp > 30: # Activar aire
elif temp < 10: # Activar calefacción
else: # Mantener estado

El código deja de ser lineal y empieza a reaccionar y adaptarse.

Control de Flujo: Automatización

Repitiendo Tareas con Bucles

Bucle `for`

Para iterar sobre una secuencia.

"Para cada nombre en mi lista, saluda."

Bucle `while`

Mientras una condición sea verdadera.

"Mientras la puerta esté abierta, sigue sonando la alarma."

`break`: Salir del bucle.

`continue`: Saltar a la siguiente iteración.

Funciones

Bloques de código reutilizables con un nombre. Las "recetas" de nuestro programa.

Módulos

Archivos con funciones que podemos `import` para usar en otros programas.

No reinventamos la rueda, ensamblamos componentes.
